Vulnérabilités OWASP Top 10 Implémentées

A01:2021 - Broken Access Control

Contrôle d'accès défaillant permettant l'accès non autorisé aux données.

GET /user/profile/{userId} - Voir n'importe quel profil sans autorisation

A02:2021 - Cryptographic Failures

Données sensibles exposées sans chiffrement approprié.

GET /user/admin/export - Export de mots de passe et données sensibles en clair

Mots de passe stockés en clair, SSN et numéros de carte de crédit non chiffrés.

A03:2021 - Injection (SQL Injection)

Vulnérabilités d'injection SQL permettant l'exécution de requêtes arbitraires.

GET /user/search?username={value} - SQL Injection via paramètre username
GET /user/list?sortBy={column} - SQL Injection via ORDER BY

Exemple: ?username=' OR '1'='1

A03:2021 - Cross-Site Scripting (XSS)

Injection de scripts malveillants dans les pages web.

GET /user/comment?username={name}&comment={text} - XSS Reflected

Exemple: ?comment=<script>alert('XSS')</script>

A04:2021 - Insecure Design (XXE)

Parser XML non sécurisé permettant l'exploitation XXE.

POST /xml/parse - XML External Entity Injection

Exemple de payload:

<?xml version="1.0"?>
<!DOCTYPE foo [<!ENTITY xxe SYSTEM "file:///etc/passwd">]>
<root>&xxe;</root>

A05:2021 - Security Misconfiguration

Configuration de sécurité inadéquate.

  • Cookies sans flags HttpOnly et Secure
  • Pas de headers de sécurité (CSP, HSTS, X-Frame-Options)
  • Upload de fichiers sans restriction de taille
  • Messages d'erreur détaillés

A06:2021 - Vulnerable and Outdated Components

Utilisation de composants avec des vulnérabilités connues.

  • Log4j 2.14.1 (CVE-2021-44228 - Log4Shell)
  • Spring Framework 5.2.0
  • Jackson 2.9.8
  • Commons FileUpload 1.3.1

A07:2021 - Identification and Authentication Failures

Authentification et gestion de session défaillantes.

POST /auth/login - Authentification faible
POST /auth/register - Inscription sans validation
POST /auth/reset-password - Réinitialisation sans vérification
  • Mots de passe stockés en clair
  • Pas de limite de tentatives de connexion
  • Messages d'erreur informatifs
  • JWT avec clé secrète faible

A08:2021 - Software and Data Integrity Failures

Désérialisation non sécurisée d'objets Java.

POST /deserialize/object - Insecure Deserialization
GET /deserialize/serialize-example?message={text} - Créer un objet sérialisé

A09:2021 - Security Logging and Monitoring Failures

Logging et monitoring insuffisants.

  • Pas de logging des tentatives de connexion
  • Pas de monitoring des activités suspectes
  • Pas d'alertes de sécurité
  • Logs insuffisants pour l'audit

A10:2021 - Server-Side Request Forgery (Bonus)

Vulnérabilités additionnelles:

GET /file/download?filename={path} - Path Traversal
POST /file/upload - Unrestricted File Upload
GET /file/read?path={path} - Arbitrary File Read
GET /file/convert?filename={file} - Command Injection

Informations de connexion par défaut

Admin: username: admin / password: admin123

Utilisateur: username: john / password: password

Outils de test recommandés

  • SAST: SonarQube, Checkmarx, Fortify, Semgrep
  • DAST: OWASP ZAP, Burp Suite, Acunetix
  • SCA: OWASP Dependency-Check, Snyk, WhiteSource
  • IAST: Contrast Security, Seeker
